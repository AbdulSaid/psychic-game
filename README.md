# psychic-game

Instructions
Click on a random letter to start the game. The computer randomly generates a letter. Its your job to guess the letter. You have 9 lives to guess it correct. If you get it right within the 9 lives, YOU WIN!. If you don't get it within the 9 lives, YOU LOSE! Get the GUESSING!
